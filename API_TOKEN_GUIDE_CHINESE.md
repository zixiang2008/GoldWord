# 🎯 CloudFlare API 令牌获取指南（中文版）

## 📋 中文界面操作步骤

### 1. 登录CloudFlare账户
- 访问 https://dash.cloudflare.com/login
- 输入邮箱和密码登录

### 2. 进入API令牌页面
- 点击右上角头像 → "我的个人资料"（My Profile）
- 点击 "API 令牌"（API Tokens）标签页
- 点击 "创建令牌"（Create Token）按钮

### 3. 选择令牌类型
- 选择 "自定义令牌"（Custom token）
- 不要使用预设模板，需要精确权限控制

### 4. 权限配置（重要！）

#### 区域权限（Zone）
```
Zone:Read      - 读取区域信息
Zone:Edit      - 编辑区域设置
```

#### 账户权限（Account）  
```
Account:Read   - 读取账户信息
```

#### R2存储权限（R2）
```
R2:Read        - 读取R2存储
R2:Write       - 写入R2存储
```

### 5. 资源范围设置
- **区域资源**（Zone Resources）: 选择 "包含所有区域"（Include All zones）
- **账户资源**（Account Resources）: 选择 "包含所有账户"（Include All accounts）
- **R2资源**（R2 Resources）: 选择 "包含所有存储桶"（Include All buckets）

### 6. 其他设置
- **TTL**: 可以设置为1年或永不过期
- **IP地址过滤**（IP Address Filtering）: 可以留空
- **账户角色**（Account Role）: 保持默认

### 7. 创建令牌
- 点击 "继续查看摘要"（Continue to summary）
- 确认权限配置正确
- 点击 "创建令牌"（Create Token）
- ⚠️ **重要**: 立即复制保存令牌！（只显示一次）

## 🚨 中文界面注意事项

### 常见翻译差异
1. **"我的个人资料"** vs **"My Profile"**
   - 位置：右上角头像下拉菜单

2. **"API 令牌"** vs **"API Tokens"**
   - 注意中间有空格

3. **"创建令牌"** vs **"Create Token"**
   - 按钮文字可能略有不同

4. **"自定义令牌"** vs **"Custom token"**
   - 选择这个选项进行详细配置

### 中文界面操作提示

1. **权限选择**
   - 在权限下拉框中输入"区域"快速找到Zone权限
   - 输入"账户"找到Account权限
   - 输入"R2"找到R2存储权限

2. **资源范围**
   - "包含所有" = "Include All"
   - 确保选择正确的资源范围

3. **确认页面**
   - 仔细核对权限列表
   - 确认资源范围设置

## 🧪 测试中文界面创建的令牌

## 🔧 问题处理

### 中文界面常见问题

**Q: 找不到"API 令牌"选项**
A: 确认已登录，点击头像后选择"我的个人资料"

**Q: 权限显示不完整**
A: 尝试刷新页面或清除浏览器缓存

**Q: 创建按钮灰色不可点击**
A: 检查是否所有必填项都已配置

**Q: 令牌创建后找不到**
A: 令牌只显示一次，创建失败需要重新创建

### 浏览器建议
- 推荐使用 Chrome、Firefox、Safari 最新版
- 关闭广告拦截插件
- 确保JavaScript已启用
